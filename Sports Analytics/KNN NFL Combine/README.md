In this project, I used the K-Nearest Neighbors Algorithm to predict NFL Combine measurable for players not participating in the combine. With nflreadr, I was able to implement up-to-date measurables from current draft prospects dating back to 2000. To clean the data, I had to adjust player position labels that had multiple names (changing OG to G, OT to T, SAF to S, etc.), quantify height ("6-2" to 72), and use one-hot encoding for position variables (1 column for each position, =1 if a player plays that position and =0 if they do not). Then, I looked at the 10 nearest neighbors of a specific player and predicted physical testing results for tests that they did not participate in based on player position, similar height/weight, and any testing they did complete. I also developed multiple visualizations to show my findings.
